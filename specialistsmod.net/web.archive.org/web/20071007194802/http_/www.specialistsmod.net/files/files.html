This file is necessary for servers using the i586 processor, such as pentiums, k6, k6-2, k6-3, first generations cyrix and the like. As a general rule, if you get an "illegal instruction" error when starting the server, then you need this other binary. Rename it as ts_i386.so and substitute the old one, or copy it to ts/dlls folder and change liblist.gam.
